The Loop Pattern

🎯 plan

Decide what experiment to run next. Which teacher? Which data? Which architecture?

⚡ execute

Run the experiment. Train the model. Generate the data. Ship the code.

📊 evaluate

Measure results against targets. Heretic exact. mk_in_ref. Token savings.

🏛 decide

SHIP, RETRAIN, PIVOT, or CONTINUE. The council reviews and decides.
